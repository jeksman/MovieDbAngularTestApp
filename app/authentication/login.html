The Movie DB App

This is an interface for The Movie DB built with AngularJS.

You don't have to be logged in to use this application but to get the most out of it you should create a Movie DB account if you don't already have one. This will allow you to rate movies and TV shows, add them to your list of favourites and create a watch list of movies and TV shows that you want to see.

Explore a little. Search for your favorite movie. Build a list of movies you want to watch.
